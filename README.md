# The-Effect-of-Personality-Traits-and-Visual-Auditory-Stimuli-on-Aesthetic-Judgments
 Investigating the impact of Big Five personality traits on multimodal (audiovisual) hedonic integration. Built with OpenSesame, this pilot study reveals a robust visual dominance effect in emotional processing, where music acts as a complementary affective lens.

Project Overview: Personality & Multimodal Integration
This study explores how we form hedonic judgments when visual and auditory stimuli are combined, and how personality traits (Five-Factor Model) moderate this process.
Key Findings:
Visual Dominance: Visual preference is the primary driver of pleasure. Multimodal stimuli showed a marginal 2.3% advantage over visual-only, confirming a "visual dominance" effect.
Personality Insights:
Conscientiousness & Openness: Correlated with generally higher hedonic ratings.
Agreeableness: Emerged as a near-significant moderator in how audiovisual harmony is perceived.
Technical Stack: Experimental paradigm designed in OpenSesame using OASIS (images) and Freesound (audio) databases.
